# Fitness Tracker

## Overview

Fitness Tracker is a full-stack web application developed to track and manage daily fitness activities. Users can add, view, update, and delete workout records easily. This project is created to practice CRUD operations using the MERN stack.

## Features

* Add new workout records
* View workout details
* Update existing workout information
* Delete workout entries
* Simple and user-friendly interface

## Tech Stack

**Frontend**

* React.js
* Axios
* HTML
* CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

## Installation and Setup

### Clone the repository

git clone https://github.com/varshini-student/fitness-tracker.git

### Navigate to the project folder

cd fitness-tracker

### Install backend dependencies

cd backend
npm install

### Run the backend server

node server.js

### Install frontend dependencies

cd ../frontend
npm install

### Run the frontend application

npm run dev

## API Operations

This project includes REST APIs to perform CRUD operations.

* **Create** – Add workout data
* **Read** – View workout records
* **Update** – Edit workout details
* **Delete** – Remove workout records
